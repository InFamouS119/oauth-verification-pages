About This Application

Online PC Store is a production-ready e-commerce mobile application built with React Native Expo. The application allows users to browse computer hardware and accessories, add items to their cart, and complete purchases.

OAuth Integration

This application uses Google OAuth for user authentication. The OAuth integration is handled through Firebase Authentication and follows all Google OAuth best practices and security guidelines.
